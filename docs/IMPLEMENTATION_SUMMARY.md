## 🔧 Quick Start

### **Development Setup**
```bash
# Clone and setup
git clone <repository>
cd windsurf-project
npm install

# Environment configuration
cp .env.example .env
# Add your API keys:
# DISCORD_TOKEN=your_bot_token
# DISCORD_CLIENT_ID=your_client_id
# GEMINI_API_KEY=your_gemini_key

# Database setup
npx prisma migrate dev
npx prisma generate

# Start development
npm run dev
```
